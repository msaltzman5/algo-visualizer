# algo-visualizer

## On fresh machine:
### Download nodejs and git (according to your operating system)
- https://nodejs.org/en/download
- https://git-scm.com/install/
### Run these commands:
- First:
  - Open a terminal  
  - Verify install  
```node -v```  
```npm -v```  
```git -v```
- If these succeed, proceed:  
```git clone https://github.com/msaltzman5/algo-visualizer.git```  
```cd algo-visualizer```  
```npm install```  
```npm start```  
- This should open up a window where you can interact with the app (no matter the operating system).
